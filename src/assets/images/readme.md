
# Image Assets Directory

This directory contains all image assets used in the Circlo application. Replace these placeholder images with your actual assets when ready for production.

## Directory Structure:
- `logos/`: App logo and partner logos
- `icons/`: UI icons and action buttons
- `rewards/`: Images related to the rewards section
- `marketplace/`: Product images for the marketplace
- `partners/`: Partner brand images and logos
- `profiles/`: Default profile images and avatars
- `sustainability/`: Images related to sustainability themes
